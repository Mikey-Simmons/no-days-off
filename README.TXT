No Days Off is a modern productivity app to track a persons personal/work/house productivity.
No Days Off uses a daily scoring system to track how productive a person is.
The app is styled using CSS and Bootstrap.
Potentially could make a daily leader board for the people who are the most productive that day/ 
This app was built using the popular Python framework Django.

Technogies used:
-Python
-HTML, CSS
-bcrypt
-Bootstrap
-github
-SQLite
-JavaScript
-jQuery
-Visual Studio Code
